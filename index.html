Github Pages

Descripción del tema

Alcances

GitHub Pages es un servicio de alojamiento web estática, está diseñado para alojar sus páginas personales, de la organización o del proyecto directamente desde un repositorio GitHub.
No es compatible con el código del lado del servidor, como, PHP, Ruby o Python
Además el protocolo usado por default para la comunicación con el servidor es HTTPS.

Limitaciones

Los repositorios de Github pages tienen un limite recomendado de 1 GB, tienen un límite de ancho de banda de 100 GB por mes.
Github Pages tiene un límite de 10 compilaciones por hora.
En caso de no seguir estas limitaciones el equipo de soporte de github envia un correo haciendo sugerencias para reducir el costo que esta tomando la pagina en los servidores.

Soporte

Github provee un soporte integrado llamado Jekyll para poder realizar una visualizacion de metadatos disponibles en el repositorio
